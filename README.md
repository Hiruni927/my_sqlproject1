🎓 School Management System

## Project Overview

This project is a School Management System that allows for efficient management of educational institution operations. It provides functionalities to track courses, students, teachers, course enrollments, and course completions. The system, built using SQL, maintains a comprehensive database of courses, their status, credits, teacher assignments, and other relevant details. It also enables effective management of students and teachers associated with the institution.

## Database Setup

Create a new database named "School" to store all the information.

##Table	Description

| Table | Description |
|-------|-------------|
| Departments | Stores department information |
| Teachers | Manages teacher/staff records |
| Students | Maintains student information |
| Courses | Stores course details |
| Enrollment | Tracks student course registrations |
| Course_Completion | Records completed courses |

## SQL Queries and Analysis

The project includes various SQL queries and data analysis tasks to extract valuable insights from the school's data. Some of the key queries implemented include:

1.Retrieve course title, category, and credits of all ACTIVE courses

2.List teacher names and their respective salaries in descending order

3.Display total count of courses in each category

4.Retrieve teacher names and designations for teachers with salary above 70,000

5.Display department numbers and total count of teachers in each department

6.Retrieve course titles containing 'Chemistry' from Courses table

7.List students who haven't completed any course yet

8.Retrieve course titles and corresponding students enrolled in those courses

9.List students who registered before 2023-01-01 and have not enrolled in any courses yet

10.Display names of students who enrolled in courses in June 2024

11.Retrieve department numbers along with count of teachers for departments having more than 2 teachers

12.Find students who have completed courses with grade 'A'

13.List all courses with their assigned teacher names

14.Show enrollment count per course

15.List highest salary per department
